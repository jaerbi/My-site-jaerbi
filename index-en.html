About me

Hello! I'm Yuri Avramenko, a web developer. I live in the city of Kalush in the Ivano-Frankivsk region. The favorite sport is cycling. Music is rock. Movies - science fiction. My favorite thing is layout sites.


At the end of 2016, I became interested in web development and plunged into her head. In order to machine the sites you need to know: HTML, CSS and JavaScript. To study these languages, I used video tutorials, various online courses, and searched for information from various sites.


Nowadays, site layout is not possible without its adabtation for different mobile devices, for this I use Flexbox, Bootstrap. Also, I use the jQuery library, which gives a good style when writing a JavaScript code. When rendering CSS code encountered a problem, the style sheet becomes scalable, so it is difficult to maintain. For this I use the preprocessor SASS. The SASS preprocessor allows you to use features that are not available in the CSS itself.
