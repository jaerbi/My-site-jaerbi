Про мене

Привіт! Я Юрій Авраменко, веб-розробник. Проживаю в місті Калуші Івано-Франківської області. Улюблений вид спорту - їзда на велосипеді. Музика - рок. Фільми - наукова фантастика. Моя улюблена справа верстка сайтів.


В кінці 2016 року зацікавився веб-розробкою і поринув в неї з головою. Для того, щоб верстати сайти необхідно знати: HTML, CSS і JavaScript. Для вивчення цих мов я користувався відео уроками, різноманітними онлайн курсами, шукав інформацію з різних сайтів.


В наш час верстка сайтів неможлива без її адабтації для різних мобільних пристроїв, для цього я використовую Flexbox, Bootstrap. Також користуюся бібліотекою jQuery, яка задає хороший стиль при написанні JavaScript-коду. При верстанні CSS коду зіткнувся з проблемою, таблиця стилів стає масштабною, тому її складно обслуговувати. Для цього я застосовую препроцесор SASS. Препроцесор SASS дозволяє використовувати функції, які недоступні в самому CSS.

Навички

Верстка сайтів по PSD-шаблонах. Адаптація сайтів.
