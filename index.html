Про мене

Привіт! Я Авраменко Юрій, веб-розробник. Проживаю в місті Калуші Івано-Франківської області. Улюблений вид спорту: їзда на велосипеді. Музика: рок. Фільми: наукова фантастика. Моя улюблен справа верстка сайтів.

В кінці 2016 року зацікавився веб-розробкою і поринув в неї з головою. Для того щоб верстати сайти необхідно знати HTML, CSS і JavaScript. Для вивчення цих мов я користувався відео уроками, різноманітними онлайн курсами, шукав інформацію з різних сайтів.

В наш час верстка сайтів не може бути без адабтації її для різних мобільних пристроїв, для цього я використовую Bootstrap. Також використовую бібліотеку jQuery вона задає хороший стиль при написанні JavaScript-коду. При написанні CSS коду зіткнувся з пробблемою, коли таблиця стилів стає величезною, то стає складно її обслуговувати. Для цього я використовую препроцесор SASS. SASS дозволяє використовувати функції які недоступні в самом CSS
