Обо мне

Здравствуйте! Я Авраменко Юрий, веб-разработчик. Проживаю в городе Калуше Ивано-Франковской области. Любимый вид спорта: езда на велосипеде. Музыка: рок. Фильмы: научная фантастика. Мое любимое дело верстка сайтов.

В конца 2016 года заинтересовался веб-разработкой и ушел в нее с головой. Для того чтобы верстать сайты необходимо знать HTML, CSS и JavaScript. Для изучения этих языков я пользовался видео уроками, разнообразными онлайн курсами, искал информацию с различных сайтов.

В настоящее время верстка сайтов не может быть без адабтации ее для различных мобильных устройств, для этого я использую Bootstrap. Также использую библиотеку jQuery она задает хороший стиль при написании JavaScript-кода. При написании CSS кода столкнулся с пробблемою, когда таблица стилей становится огромной, то становится сложно ее обслуживать. Для этого я использую препроцессор SASS. SASS позволяет использовать функции которые недоступны в самом CSS
