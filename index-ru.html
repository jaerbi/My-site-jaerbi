Обо мне

Здравствуйте! Я Юрий Авраменко, веб-разработчик. Проживаю в городе Калуше Ивано-Франковской области. Любимый вид спорта - езда на велосипеде. Музыка - рок. Фильмы - научная фантастика. Мое любимое дело верстка сайтов.


В конца 2016 года заинтересовался веб-разработкой и ушел в нее с головой. Для того, чтобы верстать сайты необходимо знать HTML, CSS и JavaScript. Для изучения этих языков я пользовался видео уроками, разнообразными онлайн курсами, искал информацию с различных сайтов.


В настоящее время верстка сайтов невозможна без ее адабтации для различных мобильных устройств, для этого я использую Flexbox, Bootstrap. Также пользуюсь библиотекой jQuery, которая задает хороший стиль при написании JavaScript-кода. При верстке CSS кода столкнулся с проблемой, таблица стилей становится масштабной, поэтому ее сложно обслуживать. Для этого я применяю препроцессор SASS. Препроцессор SASS позволяет использовать функции, которые недоступны в самом CSS.
